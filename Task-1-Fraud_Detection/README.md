# 🛡️ AI-Powered Digital Wallet Fraud Investigator

An end-to-end fraud detection and automated investigation platform designed for Pakistani Digital Wallets (e.g., EasyPaisa, JazzCash). This project combines high-speed Machine Learning for risk scoring with Generative AI for narrative reporting.

## 🚀 Key Features
- **Real-time Risk Scoring**: Instant fraud probability calculation using an optimized XGBoost model.
- **AI-Powered Investigations**: Automated, human-readable narrative reports generated by **Llama 3.3 (via Groq)** for high-risk flags.
- **Dynamic Analytics Dashboard**: Interactive visualizations of fraud trends across cities, transaction hours, and device types.
- **Feature Importance Tracking**: Transparent view of which factors (amount, velocity, location) are driving the model's decisions.
- **Pakistani Context**: Tailored to local patterns, currency (PKR), and regional city data.

## 🛠️ Tech Stack
- **Frontend**: [Streamlit](https://streamlit.io/)
- **Machine Learning**: [XGBoost](https://xgboost.readthedocs.io/), [Scikit-Learn](https://scikit-learn.org/)
- **Large Language Model**: [Groq Cloud](https://groq.com/) (Llama-3.3-70b-versatile)
- **Data Handling**: Pandas, NumPy
- **Visualizations**: Plotly Express

## 🏗️ Architecture
The system uses a **Hybrid AI Approach**:
1. **The Specialist (XGBoost)**: Analyzes numerical data (transaction amount, velocity, login attempts) to produce a precise fraud probability score.
2. **The Investigator (LLM)**: Takes the data and the score to write a professional investigation report, explaining the "Why" behind the flag.

## 📥 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mussaratshamsher/Hex-Solutions-intern.git
   cd Hex-Solutions-intern/Task-1-Fraud_Detection
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Set up environment variables**:
   Create a `.env` file in the `Task-1-Fraud_Detection` directory:
   ```env
   GROQ_API_KEY=your_api_key_here
   ```

4. **Run the application**:
   ```bash
   streamlit run src/app/main.py
   ```

## 📊 Dataset
The application utilizes a synthetic dataset of 10,000+ transactions, modeling realistic Pakistani fintech scenarios, including:
- Transaction amounts in PKR.
- Regional city distributions (Karachi, Lahore, Islamabad, etc.).
- Account age and transaction velocity metrics.

## ☁️ Deployment
This app is configured for deployment on **Streamlit Cloud**. Ensure the `GROQ_API_KEY` is added to the "Secrets" section of your Streamlit Dashboard.
